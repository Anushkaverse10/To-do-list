# Task-Tracker

A sleek and modern study task management web app designed with a Web3-inspired aesthetic.
Helps users organize tasks, track progress, and stay productive with a clean and interactive UI.

#Live Demo Link
https://to-do-list-two-rho-34.vercel.app/


##✨ Features

➕ Add new tasks instantly
✅ Mark tasks as completed
❌ Delete tasks
📊 Real-time progress tracking
💾 Data persistence using Local Storage
🎨 Modern glassmorphism UI
🌙 Light/Dark mode toggle
⚡ Smooth animations & transitions

##🛠️ Tech Stack
HTML5
CSS3 (Glassmorphism + Gradients)
JavaScript (Vanilla JS)

##📂 Project Structure
study-task-tracker/
│── index.html
│── styles.css
│── script.js
│── screenshot.png

##⚙️ How It Works
Tasks are dynamically created using JavaScript
Completed tasks are visually updated with styles
Progress bar updates based on completed vs total tasks
Local Storage ensures data stays even after refresh
